# Lowest Quality 즉시 판정 신호 목록

> 검사 카테고리 A(목적 & 유해성)와 F(스팸/남용) 수행 시 참조한다.
> 아래 신호 중 하나라도 확인되면 종합 점수와 무관하게 즉시 D등급(Lowest) 판정.

---

## 1. Harmful (유해) — 즉시 Lowest

### 1-1. 자해/타해 조장

- 자해, 자살, 섭식장애를 조장·미화·방법 제공
- 타인에 대한 물리적 폭력을 선동·지시
- 아동 착취·학대 콘텐츠

### 1-2. 특정 집단 혐오·폭력 조장

- 인종, 민족, 종교, 성별, 성적지향, 장애 등에 기반한 혐오 표현
- 특정 집단에 대한 폭력 선동
- 극단주의·테러리즘 조장

### 1-3. 유해한 오해 유발 정보

- YMYL 주제에서 전문가 합의에 명백히 반하는 정보
- 의학적으로 위험한 잘못된 건강 정보
- 선거·투표에 관한 의도적 허위 정보
- 검증 가능한 사실에 대한 명백한 거짓

---

## 2. Untrustworthy (비신뢰) — 즉시 Lowest

### 2-1. 부적절한 웹사이트/작성자 정보

- 금융 거래 사이트인데 운영자 정보·연락처가 전혀 없음
- YMYL 주제인데 작성자 자격·배경 정보가 전혀 없음
- 페이지 유형에 비해 정보가 "부적절하게" 부족 (단순 "불만족"이 아닌 수준)

### 2-2. 최하 E-E-A-T / 최하 평판

- YMYL 주제에서 전문성이 전혀 없는 작성자
- 사기, 범죄, 소비자 피해로 유명한 사이트
- 법적 제재를 받은 사이트/운영자

### 2-3. 기만적 목적·정보·디자인

- 페이지의 진짜 목적을 숨기고 다른 목적처럼 위장
- 가짜 리뷰, 조작된 추천
- 실제와 다른 제품/서비스 정보
- **기만적 디자인 유형**:
  - 광고를 MC(주요 콘텐츠)처럼 위장
  - 광고를 네비게이션 링크처럼 위장
  - 가짜 검색 결과 표시
  - 가짜 "잠금 해제" 버튼
  - 클릭베이트 제목 (MC가 제목과 무관)

### 2-4. MC 의도적 차단/은폐

- 콘텐츠가 있는 것처럼 보이지만 실제로는 접근 불가
- 의도적으로 MC를 숨기거나 제거

### 2-5. 악성 행위 의심

- 멀웨어 배포
- 피싱 시도
- 개인정보 무단 수집

---

## 3. Spammy (스팸) — 즉시 Lowest

### 3-1. 목적 불명/MC 없음/최하 MC

- 페이지 목적을 전혀 파악할 수 없음
- MC가 아예 없거나, 있어도 의미 없는 수준

### 3-2. 해킹·변조·스팸 댓글

- 사이트가 해킹당해 무관한 콘텐츠가 삽입됨
- 댓글 섹션이 스팸 링크로 도배됨

### 3-3. 만료 도메인 악용

- 만료된 도메인(정부, 학교, 자선단체 등)을 구매
- 기존 신뢰도·도메인 권위를 악용하여 무관한 콘텐츠 게시
- **탐지 신호**: 도메인 이름과 콘텐츠 주제의 극심한 불일치
  - 예: 초등학교 도메인 → 카지노 사이트
  - 예: 정부 도메인 → 제휴 마케팅

### 3-4. 사이트 평판 악용

- 1차 콘텐츠로 쌓은 사이트 랭킹을 이용해 무관한 3자 콘텐츠를 올리는 행위
- **악용인 경우**:
  - 교육 사이트에 대출 리뷰 스폰서 글
  - 의학 사이트에 "최고의 카지노" 광고 페이지
  - 뉴스 사이트에 화이트라벨 쿠폰 페이지
- **악용이 아닌 경우** (정상):
  - 와이어 서비스, 보도자료 사이트
  - 신디케이트 뉴스
  - 포럼, 댓글 등 사용자 생성 콘텐츠
  - 칼럼, 기고, 편집 콘텐츠
  - 독자를 위한 네이티브 광고 (검색 순위 조작이 아닌)
  - 적절히 처리된 제휴 링크

### 3-5. 대규모 콘텐츠 남용 (Scaled Content Abuse)

- AI나 자동화 도구로 노력·독창성·부가가치 없이 대량 생성
- **탐지 신호**:
  - 동일 템플릿으로 찍어낸 수백 개 페이지
  - 키워드만 바꾼 반복 콘텐츠
  - 문법적으로 맞지만 의미 없는 텍스트
  - 주제에 대한 깊이 없이 표면적 정보만 나열

### 3-6. 노력·독창성 없는 MC

- 의미 있는 노력 없이 만들어진 콘텐츠
- 사용자에게 아무런 가치를 제공하지 않는 콘텐츠

### 3-7. 복사 콘텐츠 (Copied Content)

- **완전 복사**: 다른 출처에서 그대로 복사·임베딩
- **최소 변형 복사**: 단어 몇 개만 바꾸거나 find-and-replace
- **동적 콘텐츠 복사**: 검색 결과·뉴스 피드 등 변동 소스에서 복사
- **패러프레이즈**: AI 도구 등으로 표현만 바꾸되 독창적 가치 없음

**주의**: 복사 자체가 자동 Lowest는 아님. MC 전체가 복사이고, 노력·독창성·부가가치가 없을 때 Lowest.

---

## 4. Low Quality (저품질) 신호 — D등급은 아니지만 감점

아래는 즉시 Lowest는 아니지만, 해당 카테고리 점수를 크게 낮추는 신호:

- 부족한 E-E-A-T (충분한 경험·전문성·권위성·신뢰성 부족)
- 저품질 MC (낮은 노력, 낮은 독창성, 낮은 부가가치)
- 필러(Filler) 콘텐츠: 의미 없는 반복·늘이기로 사용자 경험 저하
- 방해하는 광고/SC: MC를 가리거나 사용자 경험을 크게 저해
- 경미한 부정적 평판
- 불만족스러운 사이트/작성자 정보
- 과장/경미한 오해: 약간 과장되거나 오해의 소지가 있는 정보
