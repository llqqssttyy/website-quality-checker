# 점수 산출 및 등급 판정 기준

> 검사 결과를 종합 점수로 환산하고 최종 등급을 부여할 때 참조한다.

---

## 1. 카테고리별 가중치

| 카테고리                | 가중치   | 만점      | 근거                                  |
| ----------------------- | -------- | --------- | ------------------------------------- |
| A. 페이지 목적 & 유해성 | 15%      | 15점      | 기본 필터. Google QRG PQ 평가 1~2단계 |
| B. MC(주요 콘텐츠) 품질 | 25%      | 25점      | QRG에서 "가장 중요한 고려사항"        |
| C. 웹사이트/작성자 정보 | 10%      | 10점      | 신뢰 기반. YMYL일수록 중요도 상승     |
| D. E-E-A-T              | 25%      | 25점      | 핵심 프레임워크. Trust가 중심         |
| E. 평판                 | 10%      | 10점      | 외부 독립 출처 기반                   |
| F. 스팸/남용 탐지       | 15%      | 15점      | Lowest 즉시 판정 요소                 |
| **합계**                | **100%** | **100점** |                                       |

## 2. YMYL 가중치 조정

YMYL 주제로 판별된 경우, 일부 카테고리의 실질 영향력이 높아진다.
점수 산출 시 YMYL 보정 계수를 적용한다.

| 카테고리              | 비YMYL | YMYL            |
| --------------------- | ------ | --------------- |
| A. 목적 & 유해성      | 15%    | 15% (변동 없음) |
| B. MC 품질            | 25%    | 22%             |
| C. 사이트/작성자 정보 | 10%    | 13% (+3%)       |
| D. E-E-A-T            | 25%    | 28% (+3%)       |
| E. 평판               | 10%    | 12% (+2%)       |
| F. 스팸/남용          | 15%    | 10% (-5%)       |

YMYL에서는 신뢰·전문성·평판이 더 중요해지고, 스팸 탐지의 상대적 비중은 줄어든다.
(스팸이면 어차피 즉시 D등급이므로 가중치보다 즉시 판정이 우선)

## 3. 카테고리 C 페이지 유형별 차등 감점

동일한 "작성자 정보 없음"이라도 YMYL 의료 사이트와 취미 블로그에서의 심각성이 다르다. 카테고리 C에서만 아래 차등 감점을 적용한다.

| 항목    | YMYL + 상거래/정부 | 비YMYL 정보성/B2B | 비YMYL 블로그/개인 | 엔터테인먼트/유머 |
| ------- | ------------------ | ----------------- | ------------------ | ----------------- |
| C1 없음 | ❌ (-25)           | ❌ (-15)          | ⚠️ (-10)           | ⚠️ (-5)           |
| C2 없음 | ❌ (-25)           | ⚠️ (-10)          | ⚠️ (-5)            | 감점 없음         |
| C3 없음 | ❌ (-25)           | ⚠️ (-10)          | 감점 없음          | 감점 없음         |
| C4 위반 | ❌ (-25)           | ⚠️ (-15)          | ⚠️ (-12)           | ⚠️ (-10)          |

**적용 예시**:

- YMYL 의료 사이트에서 C1·C2·C3 모두 없음 → 100 - 25 - 25 - 25 = 25점
- 비YMYL B2B 기술 블로그에서 C1·C2 없음 → 100 - 15 - 10 = 75점 (❌ 1개 → 상한 70점 적용 → 70점)
- 개인 취미 블로그에서 C2 없음 → 100 - 5 = 95점

## 4. 카테고리별 점수 산출 가이드

각 카테고리는 0~100점으로 채점한 뒤, 가중치를 곱해 종합 점수에 반영한다.

### A. 페이지 목적 & 유해성 (0~100)

- 100: 명확하고 유익한 목적, 유해 요소 없음
- 70~99: 목적이 명확하나 약간의 모호함
- 40~69: 목적이 불분명하거나 약간의 과장
- 1~39: 기만적 요소가 일부 있음
- 0: 해롭거나 기만적 목적 → **즉시 D등급 트리거**

### B. MC 품질 (0~100)

- 100: 독보적 가치, 높은 독창성·노력·재능·정확성
- 75~99: 충분한 노력과 독창성, 만족스러운 콘텐츠
- 50~74: 기본은 갖추었으나 두드러지지 않음
- 25~49: 낮은 노력, 필러 콘텐츠, 부정확함
- 0~24: MC 없음, 의미 없는 콘텐츠, 전부 복사

### C. 웹사이트/작성자 정보 (0~100)

- 100: 상세한 운영자·작성자·연락처 정보
- 70~99: 충분한 정보 (페이지 유형 대비)
- 40~69: 기본 정보는 있으나 부족함
- 1~39: 불만족스럽게 부족
- 0: 부적절하게 부족 (YMYL + 정보 전무) → **즉시 D등급 트리거 가능**

### D. E-E-A-T (0~100)

- 100: 해당 분야 최고 수준 (경험+전문성+권위성+신뢰성)
- 75~99: 높은 E-E-A-T
- 50~74: 보통 수준
- 25~49: 부족한 E-E-A-T
- 0~24: 최하 E-E-A-T

### E. 평판 (0~100)

- 100: 매우 긍정적 (수상, 전문가 인정, 업계 리더)
- 70~99: 긍정적 외부 평가
- 40~49: 평판 정보 없음 — web_search 결과 정보 자체가 없는 경우 (소규모 사이트 참작)
- 35~39: 확인 불가 — URL 미제공 등으로 조사 자체가 불가능한 경우
- 25~34: 경미하게 부정적
- 0~24: 매우 부정적 (사기, 범죄) → **즉시 D등급 트리거**

### F. 스팸/남용 탐지 (0~100)

- 100: 스팸/남용 징후 전혀 없음
- 70~99: 경미한 우려 사항 1~2개
- 40~69: 일부 스팸 징후 (광고 과다, 경미한 복사 등)
- 1~39: 명확한 스팸 징후 다수
- 0: 스팸 확정 → **즉시 D등급 트리거**

## 5. 종합 점수 산출 공식

```
종합 점수 = Σ (카테고리 점수 × 가중치)
```

예시 (비YMYL):

```
A=80, B=70, C=60, D=65, E=50, F=90 인 경우:
= (80×0.15) + (70×0.25) + (60×0.10) + (65×0.25) + (50×0.10) + (90×0.15)
= 12 + 17.5 + 6 + 16.25 + 5 + 13.5
= 70.25 → B등급 (Medium)
```

## 6. 등급 판정

### 점수 기반 등급

| 종합 점수 | 등급 | Google QRG 대응 | 의미                                       |
| --------- | ---- | --------------- | ------------------------------------------ |
| 90~100    | S    | Highest         | 최고 품질. 독보적 가치 제공                |
| 75~89     | A    | High            | 고품질. 충분한 E-E-A-T와 만족스러운 MC     |
| 55~74     | B    | Medium          | 보통 품질. 목적은 달성하나 두드러지지 않음 |
| 35~54     | C    | Low             | 저품질. 개선 필요                          |
| 0~34      | D    | Lowest          | 최하 품질. 유해/비신뢰/스팸                |

### 즉시 D등급 트리거 (Override)

아래 조건 중 하나라도 해당 시, 종합 점수와 무관하게 D등급:

1. **카테고리 A = 0**: 해롭거나 기만적 목적
2. **카테고리 C = 0 + YMYL**: YMYL 주제인데 사이트/작성자 정보 전무
3. **카테고리 E ≤ 10**: 매우 부정적 평판 (사기, 범죄)
4. **카테고리 F = 0**: 스팸 확정

트리거 발동 시, 보고서에 해당 사유를 명시한다.

---

## 7. 보고서 출력 시 등급 표시

```
## 종합 등급: [S/A/B/C/D] ([점수]/100)

해당 등급: Google QRG 기준 [Highest/High/Medium/Low/Lowest]에 해당
```

D등급 트리거가 발동된 경우:

```
## 종합 등급: D (즉시 판정)

사유: [트리거 사유 명시]
원래 점수: [점수]/100 → 트리거에 의해 D등급으로 하향
```
