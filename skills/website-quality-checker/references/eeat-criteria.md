# E-E-A-T 세부 평가 기준

> 검사 카테고리 D(E-E-A-T) 수행 시 참조한다.

---

## 1. Trust(신뢰성) — 가장 중요한 핵심

페이지가 정확하고, 정직하고, 안전하고, 신뢰할 수 있는지.

### 페이지 유형별 Trust 요구 수준

| 페이지 유형       | Trust 요구 사항                                             |
| ----------------- | ----------------------------------------------------------- |
| 온라인 스토어     | 안전한 결제 시스템 + 신뢰할 수 있는 고객 서비스             |
| 제품 리뷰         | 정직하게 다른 사람의 구매 결정을 돕는 것 (판매 목적이 아닌) |
| YMYL 정보 페이지  | 정확성 필수 — 부정확하면 사람/사회에 해                     |
| 비YMYL 소셜미디어 | 높은 Trust 불필요 (오락 목적이고 해가 없으면)               |

**핵심**: 아무리 경험·전문성·권위가 있어도, 신뢰할 수 없으면 E-E-A-T는 낮음.

### Trust 판별 질문

- 페이지 정보가 사실적으로 정확한가?
- 과장·오해의 소지가 있는 표현이 없는가?
- 숨겨진 의도(판매, 유도)가 없는가?
- 안전한 결제/개인정보 처리가 보이는가? (해당 시)
- 광고와 콘텐츠가 명확히 구분되는가?

---

## 2. Experience(경험)

작성자가 해당 주제에 대한 직접적인 1인칭 체험·삶의 경험을 갖고 있는지.

### Experience 판별 질문

- 작성자가 제품/서비스/장소를 직접 사용·방문·체험한 증거가 있는가?
- 직접 촬영한 사진, 개인적 에피소드, 구체적 디테일이 포함되어 있는가?
- "직접 써봤다", "방문했다", "겪었다" 같은 1인칭 체험 서술이 있는가?
- 교과서적·일반론이 아닌 체험에서만 나올 수 있는 정보가 있는가?

### Experience가 특히 중요한 경우

- 제품 리뷰: 직접 사용한 사람 vs 스펙만 나열
- 여행 후기: 직접 방문한 사람 vs 정보만 모은 글
- 질병/건강 경험 공유: 실제 환자 vs 교과서 설명
- 레시피: 실제 만들어본 사람 vs 레시피만 복사

---

## 3. Expertise(전문성)

작성자가 해당 주제에 필요한 지식이나 기술을 갖추고 있는지.

### Expertise 판별 질문

- 작성자의 자격(학위, 인증, 직업)이 명시되어 있는가?
- 글 내용이 해당 분야의 전문 용어를 적절히 사용하는가?
- 깊이 있는 분석이나 전문적 관점이 드러나는가?
- 주제에 대한 체계적이고 정확한 설명을 제공하는가?

### 전문성 수준 기대치

| 주제        | 기대되는 전문성                  |
| ----------- | -------------------------------- |
| 의학 정보   | 의료 전문가, 의학 학위           |
| 법률 자문   | 변호사, 법학 학위                |
| 세금/재무   | 회계사, 재무 전문가              |
| 전기 배선   | 자격증 있는 전기기사             |
| 요리 레시피 | 요리 경험 (전문가 아니어도 됨)   |
| 취미 후기   | 해당 취미 경험자 (전문가 불필요) |

---

## 4. Authoritativeness(권위성)

웹사이트나 작성자가 해당 주제에서 사람들이 흔히 찾는 공식적·신뢰할 만한 출처(Go-to source)인지.

### Authoritativeness 판별 질문

- 이 사이트/작성자가 해당 주제에서 잘 알려진 출처인가?
- 다른 권위 있는 사이트에서 이 사이트를 참조·인용하는가?
- 해당 분야에서의 수상·인정·미디어 출연 등이 있는가?
- 정부/기관/학술 사이트인 경우, 해당 주제의 공식 출처인가?

### 권위성 예시

| 주제        | 권위적 출처                |
| ----------- | -------------------------- |
| 여권 갱신   | 정부 공식 사이트           |
| 약물 부작용 | FDA, 의학 저널             |
| 지역 맛집   | 지역 소셜미디어, 지역 언론 |
| 제품 스펙   | 제조사 공식 사이트         |
| 대학 입시   | 해당 대학 공식 사이트      |

---

## 5. 페이지 유형별 전문성(D2) 상한

작성자 자격이 명시되지 않은 경우, 페이지 유형에 따라 D2(Expertise) 점수 상한이 달라진다.

| 페이지 유형           | 작성자 자격 미명시 시 D2 상한 | 근거                                                       |
| --------------------- | ----------------------------- | ---------------------------------------------------------- |
| YMYL (의료·금융·법률) | 최대 40점                     | 자격 없는 YMYL 조언은 위험. QRG에서 전문성 필수로 명시     |
| 비YMYL 정보성/B2B     | 최대 70점                     | 콘텐츠 자체가 정확하고 깊이 있으면 전문성을 인정할 수 있음 |
| 비YMYL 블로그/개인    | 최대 80점                     | 일상 경험·취미 주제에서 공식 자격은 불필요                 |
| 엔터테인먼트/유머     | 상한 없음                     | 전문 자격과 무관한 영역                                    |

---

## 6. YMYL에서 Experience vs Expertise

| YMYL 주제         | Experience가 가치 있는 경우  | Expertise가 필수인 경우       |
| ----------------- | ---------------------------- | ----------------------------- |
| 임신 중 수면 문제 | 직접 겪은 사람의 비의학적 팁 | 임신 중 안전한 수면제 정보    |
| 간암 치료         | 환자·가족의 투병 경험 공유   | 치료 옵션, 기대 수명 정보     |
| 세금 신고         | 비전문가의 좌절감 표현 유머  | 실제 세금 신고 방법 안내      |
| 은퇴 저축         | 저축 서비스 사용 후기        | 얼마를 어떻게 투자해야 하는지 |
| 투표 방법         | 시민의 투표 중요성 개인 의견 | 유권자 자격, 등록 방법 정보   |

YMYL 주제에서 직접 경험 공유도 내용이 안전하고 전문가 합의와 일치하면 높은 E-E-A-T 인정.
단, YMYL 정보·조언은 전문성 필수.

---

## 7. E-E-A-T 종합 등급 판별

### 매우 높은 E-E-A-T (Highest 근거)

- 해당 분야 최고 수준의 전문가가 작성
- 풍부한 직접 경험 증거
- 업계에서 공인된 권위적 출처
- 정확하고 정직하며 사용자에게 안전

### 높은 E-E-A-T (High 근거)

- 충분한 전문성 또는 경험
- 신뢰할 만한 정보
- 긍정적 외부 평판

### 보통 E-E-A-T (Medium)

- 기본적인 역량은 있으나 두드러지지 않음

### 낮은 E-E-A-T (Low 근거)

- 주제에 비해 전문성·경험 부족
- 정보의 정확성이 의심됨
- 작성자 정보 불충분

### 최하 E-E-A-T (Lowest 근거)

- YMYL 주제에 전문성 전무
- 사기·기만적 사이트
- 매우 부정적 외부 평판
