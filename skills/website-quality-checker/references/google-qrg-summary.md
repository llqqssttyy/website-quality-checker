# Google Search Quality Rater Guidelines — 핵심 참조

> 이 문서는 검사 중 세부 판단이 필요할 때 참조한다. SKILL.md의 검사 항목만으로 판단이 어려운 경우에만 로드한다.

---

## 1. PQ 평가 3단계 흐름

1. **목적 판별**: 페이지의 진짜 목적이 해롭거나 기만적이면 → 즉시 Lowest
2. **해악 가능성 판별**: 사람/사회에 해를 끼칠 가능성, 비신뢰, 스팸 여부 → 해당 시 Lowest
3. **목적 달성도 평가**: 위 두 단계를 통과하면, 페이지가 목적을 얼마나 잘 달성하는지로 등급 결정

---

## 2. 페이지 목적 판별

**유익한 목적의 예시**:

- 정보 제공 (뉴스, 교육, 연구)
- 의견 또는 관점 공유
- 엔터테인먼트 제공
- 상품/서비스 판매
- 질문에 대한 답변 제공
- 파일 공유, 소프트웨어 다운로드
- 유머, 예술적 표현

**해로운 목적 = 즉시 Lowest**:

- 오직 돈벌이만을 위해 사용자를 돕지 않는 페이지
- 사용자를 기만하거나 해를 끼칠 목적
- 사회에 해악을 가하기 위한 페이지

## 3. YMYL 판별 기준

### YMYL이 되는 2가지 이유

1. **주제 자체가 위험**: 자해, 범죄, 폭력적 극단주의와 직접 관련
2. **부정확한 정보가 해를 끼침**: 비신뢰 출처의 정보가 건강·재정·안전에 심각한 영향

### YMYL 4대 카테고리

- **Health or Safety**: 정신적·신체적·감정적 건강, 온라인 안전
- **Financial Security**: 개인·가족의 재정적 자립을 해칠 수 있는 주제
- **Government, Civics & Society**: 공공기관 신뢰, 선거, 투표 정보
- **Other**: 기타 사회 복지에 부정적 영향

### YMYL 스펙트럼

- **명확한 YMYL**: 심장마비 증상, 투자 방법, 선거 정보 → 가장 엄격한 기준
- **중간**: 일부 주의 필요
- **명확히 아님**: 무지개의 과학, 연필 쇼핑 → 높은 정확성 불필요

## 4. MC 품질 4축

| 축                          | 설명                                                             |
| --------------------------- | ---------------------------------------------------------------- |
| **노력(Effort)**            | 사람이 적극적으로 들인 노력. 직접 제작인지 vs 자동 대량 생성인지 |
| **독창성(Originality)**     | 다른 웹사이트에 없는 고유한 정보·이미지·비디오. 원본 출처인지    |
| **재능/기술(Talent/Skill)** | 만족스러운 경험을 제공할 충분한 기술로 만들어졌는지              |
| **정확성(Accuracy)**        | 사실적 정확성과 전문가 합의(expert consensus)와의 일치           |

### 맥락별 중요도

- 의학 정보 → 정확성이 핵심
- 예술·창작 → 재능·독창성이 핵심
- How-to → 기술(Skill)이 핵심
- 소셜미디어 → 참여도·대화 깊이

### AI/자동 생성 콘텐츠

- AI 사용 자체는 품질을 결정하지 않음
- AI로 고품질 독창 콘텐츠 가능 (높은 노력 = 높은 평가)
- AI로 노력·독창성 없이 대량 찍어낸 콘텐츠 → Lowest

---

## 5. 웹사이트/작성자 정보 — 필요 수준

| 페이지 유형                 | 필요 수준                            |
| --------------------------- | ------------------------------------ |
| 금융 거래 사이트, 의료 정보 | 상세한 운영자 정보 + 고객서비스 필수 |
| YMYL 주제                   | 작성자 자격·배경 정보 필수           |
| 일반 블로그, 유머 사이트    | 닉네임/별칭만으로도 충분             |

- 페이지 유형에 비해 정보가 부적절하게 부족 → Lowest
- 불만족스럽게 부족 → Low

---

## 6. 평판 조사

### 신뢰할 수 있는 외부 출처

- 뉴스 기사, 위키백과 항목
- 전문가 추천, 수상 경력
- 상세한 고객 리뷰 (단순 별점이 아닌 구체적 내용)
- Better Business Bureau 등 소비자 보호 기관

### 평판 등급별 적용

| 평판 수준                       | PQ 등급 영향         |
| ------------------------------- | -------------------- |
| 매우 긍정적 (수상, 전문가 인정) | High → Highest       |
| 평판 정보 없음                  | Medium에서 판단 보류 |
| 경미하게 부정적                 | Low                  |
| 매우 부정적 (사기, 범죄)        | Lowest               |

### 이해관계 충돌

- 제조사가 자사 제품 리뷰 → 신뢰성 저하
- 돈 받고 쓴 인플루언서 리뷰 → 신뢰성 저하
- 실제 사용자의 솔직한 리뷰 → 신뢰성 높음

---

## 7. 특수 페이지 유형별 기준

### 백과사전 페이지

- 정확성, 출처 인용, 외부 링크 품질이 핵심

### 오류 페이지 / MC 없는 페이지

- 404 등 → 의도적으로 MC를 제거하지 않은 이상 Medium
- 의도적으로 MC 없이 만든 페이지 → Lowest

### 포럼 및 Q&A 페이지

- 참여자들의 대화 깊이와 질이 MC 품질
- 전문적이고 도움이 되는 답변 = 높은 E-E-A-T
